# todolist-angular
## *Projeto individual - Disciplina: Programação WEB II*
### Este projeto foi desenvolvido durante o 5º semestre (2024/2) da graduação em Engenharia da Computação, como trabalho prático da disciplina de Progrmação WEB II.

- Desenvolvimento de uma Single Page Application (SPA) para gerenciamento de tarefas, implementando as operações de CRUD (Criar, Ler, Atualizar, Deletar)
- Utilização de TypeScript, HTML e CSS no framework Angular para criar uma interface funcional e responsiva
-  Após o término da disciplina, o código foi revisado e a interface aprimorada com foco em organização e usabilidade.
